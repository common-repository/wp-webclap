== Description ==

This plugin can add a "Web Clap" button to your blog.
In addition, I can change a text every entry.

== Installation ==

= install from archive file. =

1. Upload 'wp-webclap' directory to the '/wp-content/plugins/' directory.
2. Change permission 'wp-webclap/caches' to 0777
3. Activate the plugin through the 'Plugins' menu in WordPress

= install from Plugin Directory. =

1. search 'WpWebclap' for plugin install page.
2. click to install "wp-webclap"
3. Activate the plugin through the 'Plugins' menu in WordPress
